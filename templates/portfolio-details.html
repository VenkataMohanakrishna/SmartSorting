About the Project

This project tackles the problem of automated fruit and vegetable sorting by identifying healthy and rotten produce. Using a Convolutional Neural Network built on MobileNetV2 with pre-trained ImageNet weights, the system accurately classifies input images.

The backend is operated securely using Flask, while the frontend gives a rich and premium user experience featuring drag-and-drop mechanics and smooth animated confidence progress bars.
